Configura Cloud Firebase

Segui questi passaggi per sincronizzare i tuoi viaggi nel cloud e condividerli con link esterni.

  1. Crea un progetto: Vai su Firebase Console e crea un progetto gratuito.
  2. Abilita Google Auth: In Authentication -> Sign-in method, attiva l'accesso tramite Google.
  3. Crea Firestore Database: Clicca su Firestore Database -> Crea database ed avvialo in modalità test.
  4. Crea Storage: In Storage, clicca su Inizia per attivare il caricamento delle immagini dei tuoi viaggi.
  5. Copia le chiavi: Nelle impostazioni del progetto, registra un'applicazione Web ed ottieni l'oggetto di configurazione firebaseConfig.
  6. Incolla nel file: Apri il file config.js nel tuo editor e inserisci le chiavi nell'oggetto window.firebaseConfig.
  7. Pubblica su GitHub: Esegui il push del file modificato. Vercel aggiornerà il tuo sito online automaticamente, abilitando il login Gmail e la sincronizzazione cloud in tempo reale!
